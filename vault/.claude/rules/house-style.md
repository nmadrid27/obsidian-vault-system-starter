# House style (example global rule)

This is an example `global`-tier rule: it returns on every prose query and acts
as a baseline style floor. Replace the contents with your own standards.

- Write in clear, direct prose. Cut filler and hedging.
- Use active voice by default.
- Prefer the shortest version that still does the job.

> Delete or rewrite this rule to match your voice. The point of the example is to
> show a working `global` rule, not to impose a style.
