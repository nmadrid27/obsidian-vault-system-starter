# Disclosure (example global rule)

This is an example `global`-tier rule for AI-collaboration disclosure. It returns
on every prose query so disclosure is never silently dropped. Replace with the
disclosure standard your institution or team uses.

- Any deliverable produced with AI assistance carries a disclosure statement
  matched to the document type.
- Do not overstate or understate the AI contribution.
- When in doubt, disclose more, not less.
